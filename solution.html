1• a – b + ab, b – c + bc, c – a + ac
(a – b + ab) + (b – c + bc) + (c – a + ac)
= a – b + ab + b – c + bc + c -a +ac
= a – a -b + b + ab – c + c + bc + ac
= ab + bc + ac
